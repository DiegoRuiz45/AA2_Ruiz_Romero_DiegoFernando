1 - Instalar Java
2 - Tener un IDE donde se pueda ejecutar spring como intellij IDEA, Spring Tools for eclipese 
    También se puede abrir con visual studio code con sus respectivos plugins de spring boot 
3 - Abrir el proyecto en algún editor.
4 - En el application.properties configurar la base de datos con los parámetros necesarios.
	Puerto de la base de datos MySQL
	Nombre de la base de datos MySQL
	Usuario
	Contraseña
4 - Ejecutar
5 - Realizar pruebas de los endpoints con la aplicaciones de postman
